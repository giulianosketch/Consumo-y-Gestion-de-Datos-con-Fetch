# Objetivos de la Aplicación
El objetivo de este proyecto es demostrar la capacidad técnica para:
Realizar peticiones asíncronas a un servicio web (API REST) utilizando `fetch` y `async/await`
Encapsular la lógica de negocio y la administración de datos dentro de una Clase de JavaScript (`UsersService`)
Implementar métodos atómicos para filtrar, buscar y ordenar información de usuarios
Vincular la lógica de programación con una interfaz de usuario dinámica mediante la manipulación del DOM
